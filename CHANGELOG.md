# Changelog

## v1.1.0 — 2026-07-23

### Новое

- **cs2-market-radar** — новый модуль: дашборд активного рынка CS2 (~1000 позиций) по рейтингам SteamDT.
  - Тянет готовый агрегат `ranking/v1/page` (по каждому предмету сразу цена, изменение цены, лоты и их изменение, сделки, оборот, популярность за 1/3/7/30 дней) вместо тысяч отдельных карточек — на порядки меньше запросов и мимо дневного лимита SteamDT.
  - **Индексы аномалий**: всплеск сделок (сегодня vs среднее за 30д), обвал лотов, скачок цены, взлёт популярности — с подсветкой, бейджами и пресетами сортировки.
  - **Каскадные фильтры** категория → редкость → износ/финиш → качество (опции сужаются под выбранное; для оружия — износ, для стикеров — Paper/Holo/Foil), переключатель периода 24ч/3д/7д/30д, KPI-плитки.
  - Сбор через SPA-скролл рейтинга (прямые запросы к ручке SteamDT подписывает/режет), панель с кнопкой пересборки, Dockerfile.

### Изменения

- **steamdt-cs2-tracker** адаптирован под редизайн SteamDT (июль 2026): «продано сегодня» считается из ряда `type-trend` вместо умершей ручки `overview/today`; увеличен дедлайн карточки (18→38с) от точечных таймаутов; панель считает завершение по реально сохранённым позициям (недобранное добирается авто-возобновлением).

> Примечание: у SteamDT есть суточный лимит запросов на IP для анонимных пользователей — поштучный сбор всех 1507 пушек в `steamdt-cs2-tracker` в него упирается. `cs2-market-radar` этот лимит обходит по конструкции (берёт агрегат), а для трекера решение — вход в аккаунт SteamDT (выше потолок).

## v1.0.0 — 2026-07-18

Первый единый релиз монорепозитория: три инструмента собраны вместе, проверены на запуск с чистого клона (Windows/Linux/macOS и Docker).

### Новое

- **lisskins-steamdt-compare** — новый инструмент: сравнение цен избранного списка lis-skins против Steam / BUFF / YouPin (через SteamDT).
  - Интерактивная таблица: поиск, сортировка, фильтры по категории и редкости (редкости зависят от выбранной категории), подсветка самой дешёвой площадки, ссылки на предмет на lis-skins и SteamDT (страница с графиком трендов).
  - Пересборка данных по кнопке на локальной панели (`http://localhost:8317`): headless-Chromium сам собирает свежие цены и перезаписывает `index.html` / `comparison.xlsx` / `data.json` на диске; при запуске вне Docker — авто-коммит и пуш данных.
  - Выгрузка в Excel с формулами (курсы валют, разница в ₽ и %).

### Изменения и исправления

- **Монорепозиторий**: `steamdt-cs2-tracker` и `cs2-china-stats` перенесены сюда с полной историей коммитов; старые отдельные репозитории удалены.
- **Обход антибота lis-skins**: пагинация списка идёт кликами внутри SPA с перехватом её API-ответов — полные перезагрузки страниц ловила Cloudflare-заглушка, непроходимая в headless.
- **Рейт-лимит SteamDT**: пауза между запросами и 65-секундный бэкофф при «too fast» — сбор доходит до конца без потерянных позиций.
- **Docker для обоих Node-проектов**: готовые Dockerfile на базе образов Playwright; compare-контейнер сам доустанавливает зависимости, если папка примонтирована с чистого клона.
- **Переносимость трекера**: панель уважает внешний `PLAYWRIGHT_BROWSERS_PATH` (работает в Docker), остановка сбора и открытие отчёта кроссплатформенные; при краш-лупе панель показывает настоящую причину вместо 15 молчаливых перезапусков.
- **Свежий клон «просто работает»**: `npm install`/`npm ci` сами ставят Chromium (postinstall), `.bat`-панели ставят зависимости при первом запуске, понятные сообщения об отсутствии зависимостей вместо стеков ошибок.
- **README** во всех проектах: таблица требований, инструкции для Windows / Linux / macOS / Docker, известные ограничения.

### Как обновиться

```
git pull
```
Для `lisskins-steamdt-compare` и `steamdt-cs2-tracker` после обновления достаточно перезапустить панель (`Открыть панель.bat` / `node server.mjs` / `node ui_server.mjs`) — недостающие зависимости поставятся сами. Docker-образы пересоберите: `docker build ...` (см. README проектов).
