# lisskins-steamdt-compare

Сравнение цен избранных скинов CS2 с [lis-skins.com](https://lis-skins.com) против Steam, BUFF и YouPin (данные через [SteamDT](https://www.steamdt.com)).

## Файлы

- `index.html` — интерактивный отчёт (поиск, сортировка, подсветка самой дешёвой площадки). Открывается двойным кликом в любом браузере.
- `comparison.xlsx` — та же выгрузка в Excel с формулами (курсы валют и разница пересчитываются при правке жёлтых ячеек).

## Как обновить данные

Оба файла — снимок на момент сбора (дата и курсы валют указаны в шапке). Автоматического обновления нет: и lis-skins, и SteamDT блокируют прямые запросы к своим API (антибот), поэтому сбор данных идёт через управляемый браузер (Claude Code), имитирующий реальные клики и ввод текста в поиск.

Чтобы обновить:
1. Откройте `index.html`, вставьте нужную ссылку на список избранного lis-skins (`user_list_id=...`) в поле сверху.
2. Нажмите «Скопировать запрос для Claude» — готовый промпт скопируется в буфер обмена.
3. Вставьте его в чат Claude Code — Claude заново соберёт данные, **перезапишет этот `index.html` и `comparison.xlsx` прямо в этой папке** (не только покажет таблицу в чате) и запушит изменение в репозиторий. Если файл был открыт в браузере — перезагрузите вкладку, чтобы увидеть новые данные.

## Как это устроено

- Список избранного lis-skins вытаскивается из клиентского состояния страницы (vue-query кэш, ключ `userLists`) — интерфейс сайта не фильтрует выдачу по списку из-за бага, поэтому данные читаются напрямую из кэша.
- Цены SteamDT приходят из ответа `/api/skin/market/v3/page` в юанях (CNY); курс CNY→RUB берётся из `localStorage.commonDictionary.rates` того же сайта.
