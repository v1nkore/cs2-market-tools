# CS2 Market Tools

Набор независимых инструментов для отслеживания и сравнения цен и статистики по скинам CS2 на разных площадках (lis-skins, Steam, BUFF, YouPin/悠悠, агрегатор SteamDT). Каждый инструмент — отдельная папка со своим README, можно использовать по отдельности.

## Проекты

### [`lisskins-steamdt-compare/`](lisskins-steamdt-compare/)

Сравнение цен избранного списка скинов с lis-skins.com против Steam, BUFF и YouPin (данные — через агрегатор SteamDT). Интерактивная HTML-таблица с поиском, сортировкой и подсветкой самой дешёвой площадки, плюс выгрузка в Excel. Пересборка данных — по кнопке на локальной панели: headless-Chromium сам собирает свежие цены и перезаписывает `index.html`/`comparison.xlsx`/`data.json` на диске.

**Стек:** Node.js + Playwright (сборщик и панель) + статический HTML-отчёт.

**Запуск (Windows, проще всего):**
```
двойной клик по «Открыть панель.bat»
```
При первом запуске сам поставит зависимости и Chromium, затем откроет панель на http://localhost:8317.

**Запуск вручную:**
```
cd lisskins-steamdt-compare
npm install
npx playwright install chromium
node server.mjs          # панель на http://localhost:8317
```

**Запуск в Docker:**
```
cd lisskins-steamdt-compare
docker build -t lisskins-steamdt-compare .
docker run -d --name lisskins-compare -p 8317:8317 -v "%cd%:/app" lisskins-steamdt-compare   # Windows cmd
# в PowerShell вместо %cd% используйте ${PWD}, в bash — $(pwd)
```
Том `-v` монтирует папку проекта, чтобы пересборка перезаписывала файлы на хосте, а не только внутри контейнера. Git-пуш внутри контейнера отключён.

**Использование:** на странице вставьте ссылку на список избранного lis-skins (`?user_list_id=...`) и нажмите «Пересобрать по ссылке». Сборка ~10 минут с прогрессом; по завершении файлы на диске перезаписаны, страница перезагружается сама. Готовый `index.html` можно открывать и просто как файл — без сервера кнопка работает в запасном режиме (копирует промпт для Claude Code). Пересборка из консоли без панели: `node scrape.mjs --url "<ссылка>"`.

### [`steamdt-cs2-tracker/`](steamdt-cs2-tracker/)

Трекер показателя «продано сегодня» (Sold Today) для наклеек киберспортивных команд и популярных пушек — данные снимаются со SteamDT, копится история по дням, строятся HTML-отчёты (дашборд, сводная матрица по дням, снимок за конкретный день).

**Стек:** Node.js + Playwright (headless Chromium — сайт блокирует прямые HTTP-запросы антиботом, поэтому сбор идёт через настоящий браузер).

**Установка (один раз):**
```
cd steamdt-cs2-tracker
npm ci
set PLAYWRIGHT_BROWSERS_PATH=%cd%\pw-browsers
npx playwright install chromium
```

**Запуск (Windows):**
```
двойной клик по «Открыть панель.bat»
```
Откроется панель управления на http://localhost:4317 — там кнопки запуска/остановки сбора для наклеек и пушек, прогресс-бары и ссылки на отчёты. Окно консоли во время работы не закрывать.

**Запуск из консоли:**
```
node scrape.mjs                      # полный сбор наклеек
node scrape.mjs --tournament austin  # только один турнир
node report.mjs                      # пересобрать HTML-отчёты
node scrape_guns.mjs                 # сбор пушек
```

**Запуск в Docker:**
```
cd steamdt-cs2-tracker
docker build -t steamdt-cs2-tracker .
docker run -p 4317:4317 -v "%cd%/data:/app/data" -v "%cd%/reports:/app/reports" steamdt-cs2-tracker
```
Тома для `data/` и `reports/` — чтобы собранная история и отчёты не терялись между перезапусками контейнера.

Собранные данные (`data/`, `data_guns/`, отчёты) в git не хранятся — история едет между машинами только через синхронизацию папки. Подробности — в README проекта.

### [`cs2-china-stats/`](cs2-china-stats/)

Дашборд статистики по предметам CS2 из инвентарей отслеживаемых китайских инвесторов: таблица с количеством и долей от общего числа, разбивка по 20 категориям, топ-15 предметов, поиск и фильтры. Данные — встроенный снимок из публичной Google-таблицы с кнопкой обновления «на лету» (запрос идёт напрямую в Google-таблицу и работает даже из `file://`).

**Стек:** один самодостаточный HTML-файл, без сборки и зависимостей.

**Запуск:**
```
открыть cs2-china-stats/index.html двойным кликом в любом браузере
```
Никакой установки не требуется. Можно развернуть на GitHub Pages (Settings → Pages → Deploy from a branch) — файл в корне папки обслуживается как главная страница.

## Общее

- Все три инструмента так или иначе работают со SteamDT и/или lis-skins — оба сайта блокируют прямые API-запросы антиботом, поэтому сбор данных везде идёт либо через управляемый браузер (Playwright, реальный Chromium), либо использует источники, изначально открытые для чтения (публичная Google-таблица).
- `lisskins-steamdt-compare/` содержит конкретный список избранного и цены на момент сбора — это персональные данные, учитывайте это при дальнейшем шаринге.
- История коммитов из исходных отдельных репозиториев (`steamdt-cs2-tracker`, `cs2-china-stats`) сохранена при объединении в этот монорепозиторий.
