# CS2 Market Tools

Инструменты для отслеживания и сравнения цен на скины CS2. Три проекта в одном репозитории:

- [`lisskins-steamdt-compare/`](lisskins-steamdt-compare/) — сравнение цен избранного с lis-skins против Steam/BUFF/YouPin (через SteamDT).
- [`steamdt-cs2-tracker/`](steamdt-cs2-tracker/) — трекер дневных продаж скинов CS2 (Node + Playwright).
- [`cs2-china-stats/`](cs2-china-stats/) — статистика по китайским площадкам CS2.

Каждый проект — самостоятельный, со своим README внутри папки.
