# CS2 · Инвентари китайских инвесторов — статистика по предметам

Интерактивный дашборд со статистикой по предметам CS2 из отслеживаемых портфолио китайских инвесторов.
Один самодостаточный HTML-файл: открывается двойным кликом или разворачивается на GitHub Pages.

> **Live demo:** https://v1nkore.github.io/cs2-investors-stats/
> _(станет доступен после включения GitHub Pages — см. раздел «Публикация»)_

---

## Возможности

- **Таблица предметов** — название, количество, доля от общего числа и кнопка **SteamDT** для быстрого перехода к актуальной цене (`steamdt.com/cs2/<предмет>`).
- **KPI**: уникальных позиций, суммарное количество штук, число категорий, самый массовый предмет.
- **Разбивка по 20 категориям** (оружие, StatTrak™, ножи/перчатки, стикеры, чармы, граффити, кейсы, медали, …) с переключателем «по сумме штук / по числу позиций». Клик по категории фильтрует таблицу.
- **Топ-15 предметов** — адаптируется под выбранную категорию.
- Поиск по названию, фильтр по категории, сортировка, пагинация (100/стр.).
- **Кнопка «Обновить данные»** — подтягивает свежий срез напрямую из Google-таблицы (gviz JSONP, обходит CORS, работает даже из `file://`). Если сети нет — используется встроенный снапшот.

## Данные

- Источник: публичная Google-таблица, лист **«Item counts»** (`gid=721854734`), колонки `Item Name / Count / Category`.
- В `index.html` встроен снапшот на **12.07.2026** (~28 600 позиций) — страница работает мгновенно и офлайн.
- «Обновить данные» заменяет снапшот живым срезом из таблицы на момент нажатия.

## Запуск локально

Просто откройте `index.html` в браузере. Всё (данные, стили, скрипты) — внутри файла, зависимостей и сборки нет.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и запушьте содержимое этой папки (см. ниже).
2. **Settings → Pages → Build and deployment → Source: `Deploy from a branch`**, ветка `main`, папка `/ (root)`.
3. Через ~1 минуту сайт будет доступен по адресу https://v1nkore.github.io/cs2-investors-stats/.

`index.html` в корне обслуживается GitHub Pages как главная страница автоматически.

## Оговорки

- Количество отражает **массовость** предмета, а не его стоимость: в топе — кейсы и дешёвые стикеры. Цену смотрите по кнопке SteamDT.
- Данные агрегированы из внешнего источника (в основном csfloat) и приблизительны: ваниллы, приватные аккаунты и складские ячейки представлены неполно.
